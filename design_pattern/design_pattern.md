# 디자인 패턴

### 디자인 패턴은 클래스 라이브러리가 아니다.

디자인 패턴은 더 일반적인 개념으로 클래스 라이브러리는 부품이 된 프로그램이고, 디자인 패턴은 부품이 어떻게 조립되어있는지, 개별 부품이 어떻게 관련되어 큰 기능을 하는지 표현한 것이다. 

즉 클래스 끼리 관계를 나타내는 큰 개념이라고 생각하면 될 듯 싶다.
